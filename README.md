# Description

Django dnsmasq is a Django application to create configuration files
for dnsmasq.

# System Requirements

* Python 3.x
* Django 3.2.3 (https://pypi.org/project/Django/)
* django-macaddress (https://pypi.org/project/django-macaddress/)
